OPTION 2 — "DEVELOPER TERMINAL"

Design Philosophy: Leans into Jeffrey's identity as a developer. Monospace typography, code-comment section headers, terminal-window hero, file-explorer project layout. Still warm and readable — not dark/hacker themed, just distinctly "built by a programmer."

Key Design Differences vs. Current Site:
  • Two-column layout: persistent sidebar navigation + scrollable content area
  • Terminal window hero — types out bio on load (Framer Motion)
  • Section headers as code comments: // experience, /* projects */
  • Projects shown as a file-explorer grid (folders/files metaphor)
  • Education, Skills, Side Quests in 3-column footer row
  • Skills shown as terminal output (not chip grid)
  • Footer is a terminal prompt line
